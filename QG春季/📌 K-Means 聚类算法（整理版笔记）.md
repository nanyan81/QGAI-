# 📌 K-Means 聚类算法（整理版 · GitHub优化）

------

## 一、基本概念

K-Means 是一种基于**欧式距离**的无监督学习聚类算法，其核心思想是：

> 两个样本之间的距离越近，相似度越高

------

## 二、算法思想（牧师-村民模型）

为了更直观理解 K-Means，可以用如下类比：

- K 个牧师 → 聚类中心（centroid）
- 村民 → 数据点（samples）

### 过程如下：

1. 初始时，随机选择 K 个牧师的位置
2. 每个村民选择离自己最近的牧师
3. 每个牧师移动到其“负责的村民”的中心位置
4. 重复上述过程，直到牧师位置不再变化

👉 最终：算法收敛

------

## 三、算法步骤（核心）

### Step 1：初始化

选择 K 个初始样本作为聚类中心：

```text
a = a_1, a_2, ..., a_k
```

------

### Step 2：样本分配

对于每个样本 $x_i$：

- 计算其到各个聚类中心的距离
- 分配到最近的聚类中心对应的类别

------

### Step 3：更新聚类中心

对于每个类别 $C_j$：

$$
a_j = \frac{1}{|C_j|} \sum_{x \in C_j} x
$$

其中：

- $C_j$：第 j 个簇
- $|C_j|$：该簇样本数量
- $x$：簇内样本

------

### Step 4：迭代

重复 Step 2 和 Step 3，直到满足停止条件：

- 聚类中心不再变化
- 或达到最大迭代次数

------

## 四、目标函数（核心公式）

K-Means 的优化目标是最小化簇内平方误差（SSE）：

$$
\sum_{i=1}^{k} \sum_{x \in C_i} ||x - \mu_i||^2
$$

------

## 五、伪代码实现

```python
# 输入：数据集 X（n 个样本，每个 m 维），簇数 k
# 输出：k 个聚类中心

初始化 k 个聚类中心

while 未收敛:
    # 分配步骤
    for i in range(n):
        计算 x_i 到所有中心的距离
        将 x_i 分配给最近的中心

    # 更新步骤
    for j in range(k):
        取出属于第 j 类的所有样本
        更新中心为这些样本的均值
```

------

## 六、复杂度分析

### 1️⃣ 时间复杂度

$$
O(t \cdot k \cdot n \cdot m)
$$

- $t$：迭代次数
- $k$：簇数
- $n$：样本数
- $m$：特征维度

------

### 2️⃣ 空间复杂度

$$
O(m (n + k))
$$

------

## 七、优缺点分析

### ✅ 优点

1. 原理简单，易理解
2. 实现方便
3. 计算效率较高

------

### ❌ 缺点

1. 对初始中心敏感（易陷入局部最优）
2. 对异常值（离群点）敏感
3. 需要预先指定 K 值
4. 不适合非凸形状数据

------

## 八、常见优化方法

1. **K-Means++ 初始化**（减少随机性）
2. **数据标准化**（避免量纲影响）
3. **肘部法则（Elbow Method）**选择 K
4. **Mini-Batch K-Means**（加速大数据计算）

------

## 九、Python 实现示例（sklearn）

```python
from sklearn.cluster import KMeans
import numpy as np

X = np.array([
    [1, 2], [1, 4], [1, 0],
    [10, 2], [10, 4], [10, 0]
])

kmeans = KMeans(n_clusters=2, random_state=0)
kmeans.fit(X)

print("标签:", kmeans.labels_)
print("中心:", kmeans.cluster_centers_)
```

------

## 🎯 一句话总结

> K-Means = 不断“分组 + 更新中心”，直到结果稳定的聚类算法
